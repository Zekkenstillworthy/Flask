Welcome to RiddleNet! This is a simple web application that demonstrates how to implement two-factor authentication (2FA) using Flask and PyOTP. You can sign up for an account, log in, and view your profile. The application uses a secret key to generate a one-time password (OTP) that you can use to authenticate yourself. You can also generate a QR code that you can scan with a 2FA app like Google Authenticator or Authy.

System Overview

The system consists of several key components:

  • User Authentication: Users can sign up and log in using their credentials. The system uses hashed passwords for security.
  • Two-Factor Authentication (2FA): After logging in, users are required to enter a one-time password (OTP) generated by a 2FA app. This adds an extra layer of security.
  • Profile Management: Users can update their profile information, including their username, password, and profile image.
  • Quiz Functionality: Users can participate in quizzes, and their scores are saved and displayed on a leaderboard.
  • Leaderboard: The leaderboard displays the highest scores of all users, along with the date of their last attempt.
  • Score Management: Users can view and delete their past quiz scores.

How It Works

Here is a step-by-step explanation of how the system works:

  1. Sign Up: Users create an account by providing a username and password. The password is hashed and stored securely in the database.
  2. Log In: Users log in using their credentials. If the credentials are correct, they are prompted to enter an OTP generated by their 2FA app.
  3. Generate QR Code: When a user signs up, a secret key is generated and used to create a QR code. The user scans this QR code with their 2FA app to set up 2FA.
  4. Verify OTP: During login, the user enters the OTP from their 2FA app. The system verifies the OTP and grants access if it is correct.
  5. Dashboard: After logging in, users are redirected to the dashboard where they can start quizzes, view their profile, and check the leaderboard.
  6. Quiz Participation: Users can start a quiz, answer questions, and submit their answers. The system calculates their score and saves it in the database.
  7. View and Delete Scores: Users can view their past quiz scores and delete any score they wish to remove.

Crimping Simulation

The UTP Cable Crimping Simulation allows users to practice crimping cables virtually. Users can select the wiring type (Straight-Through or Crossover), drag and drop wires into the correct slots, and check their wiring.

  1. Select Wiring Type: Choose between Straight-Through (T568B) or Crossover (T568A to T568B).
  2. Arrange Wires: Drag and drop the wires into the correct slots for both ends of the cable.
  3. Check Wiring: Click the "Crimp the Cable" button to verify the wiring. If correct, a success message is displayed; otherwise, users can try again.

Network Topology

The Network Topology tool allows users to create and verify different network topologies. Users can drag and drop devices onto a canvas, connect them, and check if the topology is correct.

  1. Select Topology: Choose a topology from the dropdown menu.
  2. Drag and Drop Devices: Drag devices from the palette to the canvas.
  3. Connect Devices: Click "Connection Mode" and connect devices by clicking on them.
  4. Check Topology: Click "Check Topology" to verify if the created topology matches the selected one.

Network Troubleshooting Guide

The Network Troubleshooting Guide helps users learn how to troubleshoot common network issues. Users can select a problem, drag and drop devices, and simulate troubleshooting scenarios.

  1. Select Problem: Choose a network problem from the dropdown menu.
  2. Drag and Drop Devices: Drag devices from the palette to the canvas.
  3. Connect Devices: Click "Connection Mode" and connect devices by clicking on them.
  4. Check Scenario: Click "Check Scenario" to verify if the setup addresses the selected problem.
